## 📚 Projects Overview

This repository contains all projects from the **Back-End Web Development** specialization at Holberton School. Each project builds on the previous one, covering essential concepts in modern back-end development.

### 🐍 Python

| 📂 Project | 🎯 Description | 🛠️ Technologies |
|-----------|---------------|-----------------|
| [**Python Variable Annotations**](./python_variable_annotations) | Type annotations in Python 3 - type hints, duck typing, and code validation with mypy | Python 3, mypy |
| [**Python Async Function**](./python_async_function) | Asynchronous programming - async/await syntax, coroutines, and concurrent execution | Python 3, asyncio |
| [**Python Async Comprehension**](./python_async_comprehension) | Async generators and comprehensions - collecting data from async sources efficiently | Python 3, asyncio |
| [**Pagination**](./pagination) | Paginating datasets - simple pagination, hypermedia pagination, and deletion-resilient pagination | Python 3, REST API |
| [**Caching**](./caching) | Caching strategies - FIFO, LIFO, LRU, MRU, and LFU cache replacement policies | Python 3, OOP |
| [**Personal Data**](./personal_data) | Handling PII - log filtering, password encryption, and database authentication | Python 3, bcrypt, MySQL |
| [**Basic Authentication**](./Basic_authentication) | HTTP Basic Authentication - base64 encoding, API authentication, and error handling | Python 3, Flask, REST API |
| [**Session Authentication**](./Session_authentication) | Session-based auth - cookies, session IDs, and user session management | Python 3, Flask, REST API |
| [**User Authentication Service**](./user_authentication_service) | Full auth service - user registration, login, sessions, and password reset with Flask | Python 3, Flask, SQLAlchemy, bcrypt |
| [**Unittests and Integration Tests**](./Unittests_and_integration_tests) | Testing in Python - unittest, mocking, parameterized tests, and integration testing | Python 3, unittest, parameterized |
| [**i18n**](./i18n) | Internationalization - Flask-Babel, locale detection, timezone handling, and translations | Python 3, Flask, Flask-Babel |

### 📦 JavaScript / Node.js

| 📂 Project | 🎯 Description | 🛠️ Technologies |
|-----------|---------------|-----------------|
| [**ES6 Basic**](./ES6_basic) | Modern JavaScript fundamentals - let/const, arrow functions, template literals, spread/rest operators | JavaScript ES6, Node.js |
| [**ES6 Promises**](./ES6_promise) | Asynchronous JavaScript - Promises, then/catch/finally, async/await, and error handling | JavaScript ES6, Node.js |
| [**ES6 Classes**](./ES6_classes) | Object-oriented JavaScript - classes, inheritance, static methods, and metaprogramming | JavaScript ES6, Node.js |
| [**ES6 Data Manipulation**](./ES6_data_manipulation) | Advanced data structures - Map, Set, WeakMap, typed arrays, and array methods | JavaScript ES6, Node.js |
| [**Node.js Basic**](./Node_JS_basic) | Node.js fundamentals - modules, file system, HTTP server, and Express.js basics | Node.js, Express.js |
| [**Unittests in JS**](./unittests_in_js) | JavaScript testing - Mocha, Chai, Sinon, stubs, spies, hooks, and integration testing | Node.js, Mocha, Chai, Sinon |
| [**Queuing System in JS**](./queuing_system_in_js) | Job queues with Redis and Kue - async job processing, tracking, and progress management | Node.js, Redis, Kue, Express.js |

### 🗄️ Databases

| 📂 Project | 🎯 Description | 🛠️ Technologies |

| [**MySQL Advanced**](./MySQL_Advanced) | Advanced SQL - stored procedures, triggers, views, indexes, and query optimization | MySQL |
| [**NoSQL**](./NoSQL) | MongoDB fundamentals - CRUD operations, aggregation pipeline, and Python integration with PyMongo | MongoDB, Python, PyMongo |
| [**Redis Basic**](./0x0B_redis_basic) | Redis fundamentals - caching, data storage, expiration, and using Redis as a simple data store with Python | Redis, Python |

## 🎓 Learning Objectives

Throughout these projects, you will master:

- ✅ **Python Back-End** - Type annotations, async programming, and API development
- ✅ **Authentication** - Basic auth, session auth, and full user authentication services
- ✅ **Modern JavaScript** - ES6+, Node.js, and Express.js
- ✅ **Databases** - MySQL, MongoDB, and Redis
- ✅ **Testing** - Unit tests, integration tests, mocking, and TDD
- ✅ **Caching & Queuing** - Cache strategies and job queue management
- ✅ **Security** - PII handling, encryption, and secure API design
- ✅ **i18n** - Internationalization and localization
