## 📖 Description

This project introduces ES6 (ECMAScript 2015) features. You will learn the differences between `var`, `let`, and `const`, arrow functions, template literals, spread/rest operators, and more.

## 🎯 Learning Objectives

- What ES6 is and its new features
- The difference between a constant and a variable
- Block-scoped variables
- Arrow functions and default function parameters
- Rest and spread function parameters
- String templating in ES6
- Object creation and their properties in ES6
- Iterators and for-of loops

## 🛠️ Technologies

- JavaScript ES6
- Node.js
- Babel
- ESLint
