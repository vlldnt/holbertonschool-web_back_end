## 📖 Description

This project introduces NoSQL databases with MongoDB. You will learn the differences between SQL and NoSQL, how to perform CRUD operations, and how to use MongoDB with Python through PyMongo.

## 🎯 Learning Objectives

- What NoSQL means and the difference between SQL and NoSQL
- What ACID is
- What document storage is
- NoSQL types
- How to query, insert, update, and delete information from a NoSQL database
- How to use MongoDB with Python (PyMongo)

## 🛠️ Technologies

- MongoDB
- Python 3
- PyMongo
