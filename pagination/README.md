## 📖 Description

This project covers different pagination techniques for REST APIs. You will implement simple page-based pagination, hypermedia pagination with metadata, and deletion-resilient pagination.

## 🎯 Learning Objectives

- How to paginate a dataset with simple `page` and `page_size` parameters
- How to paginate a dataset with hypermedia metadata
- How to paginate in a deletion-resilient manner

## 🛠️ Technologies

- Python 3
- REST API design
