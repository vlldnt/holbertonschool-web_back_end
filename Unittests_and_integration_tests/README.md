## 📖 Description

This project covers unit testing and integration testing in Python. You will learn the difference between unit and integration tests, common testing patterns like mocking, parametrization, and fixtures.

## 🎯 Learning Objectives

- The difference between unit and integration tests
- Common testing patterns: mocking, parametrizations, and fixtures
- How to use `unittest.mock` to mock external calls
- How to use `parameterized` for parametrized tests

## 🛠️ Technologies

- Python 3
- unittest
- parameterized
- unittest.mock
