## 📖 Description

This project covers internationalization (i18n) and localization (l10n) in a Flask application. You will learn to parametrize templates for multiple languages, infer locale from request parameters, and localize timestamps.

## 🎯 Learning Objectives

- How to parametrize Flask templates to display different languages
- How to infer the correct locale based on URL parameters, user settings, or request headers
- How to localize timestamps

## 🛠️ Technologies

- Python 3
- Flask
- Flask-Babel
- pytz
