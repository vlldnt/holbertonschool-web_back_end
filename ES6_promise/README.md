## 📖 Description

This project covers Promises in JavaScript. You will learn how to create and use Promises, handle async operations with `then`, `catch`, and `finally`, and use `async/await` syntax.

## 🎯 Learning Objectives

- Promises (how, why, and what)
- How to use the `then`, `resolve`, `catch` methods
- How to use every method of the Promise object
- The `throw` / `try` / `catch` pattern
- The `await` operator and `async` functions

## 🛠️ Technologies

- JavaScript ES6
- Node.js
- Babel
- ESLint
