## 📖 Description

This project introduces asynchronous programming in Python using `async` and `await` syntax. You will learn to write concurrent coroutines, measure execution time, and create async tasks.

## 🎯 Learning Objectives

- `async` and `await` syntax
- How to execute an async program with `asyncio`
- How to run concurrent coroutines
- How to create asyncio tasks
- How to use the `random` module

## 🛠️ Technologies

- Python 3
- asyncio
