## 📖 Description

This project covers testing in JavaScript. You will learn how to use Mocha for test suites, Chai for assertions, Sinon for spies and stubs, and how to write integration tests for a Node.js API.

## 🎯 Learning Objectives

- How to use Mocha to write a test suite
- How to use different assertion libraries (Node assert, Chai)
- How to present long test suites
- When and how to use spies and stubs (Sinon)
- What are hooks and when to use them
- Unit testing with async functions
- How to write integration tests with a small Node server

## 🛠️ Technologies

- Node.js
- Mocha
- Chai
- Sinon
- Express.js
