## 📖 Description

This project covers advanced MySQL concepts. You will learn to create tables with constraints, optimize queries with indexes, and implement stored procedures, functions, views, and triggers.

## 🎯 Learning Objectives

- How to create tables with constraints
- How to optimize queries by adding indexes
- What stored procedures and functions are and how to implement them
- What views are and how to implement them
- What triggers are and how to implement them

## 🛠️ Technologies

- MySQL 5.7
