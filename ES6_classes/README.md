## 📖 Description

This project covers ES6 classes in JavaScript. You will learn how to define classes, add methods, use static methods, extend classes with inheritance, and explore metaprogramming with symbols.

## 🎯 Learning Objectives

- How to define a class
- How to add methods to a class
- Why and how to add a static method to a class
- How to extend a class from another
- Metaprogramming and symbols

## 🛠️ Technologies

- JavaScript ES6
- Node.js
- Babel
- ESLint
