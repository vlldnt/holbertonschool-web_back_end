## 📖 Description

This project explores different caching algorithms and replacement policies. You will implement FIFO, LIFO, LRU, MRU, and LFU caching systems inheriting from a base caching class.

## 🎯 Learning Objectives

- What a caching system is
- What FIFO, LIFO, LRU, MRU, and LFU mean
- The purpose and limits of a caching system

## 🛠️ Technologies

- Python 3
- OOP (Inheritance)
