## 📖 Description

This project covers data manipulation using ES6 data structures. You will learn how to use `map`, `filter`, and `reduce` on arrays, as well as typed arrays, Sets, Maps, and WeakMaps.

## 🎯 Learning Objectives

- How to use `map`, `filter`, and `reduce` on arrays
- Typed arrays
- The Set, Map, and WeakMap data structures

## 🛠️ Technologies

- JavaScript ES6
- Node.js
- Babel
- ESLint
