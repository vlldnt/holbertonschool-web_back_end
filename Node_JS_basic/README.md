## 📖 Description

This project introduces Node.js fundamentals. You will learn how to run JavaScript with Node.js, use the process module, read files, create HTTP servers, and build a small application with Express.js.

## 🎯 Learning Objectives

- How to run JavaScript using Node.js
- How to use Node.js modules
- How to read files using a specific Node.js module
- How to access command line arguments and the environment using `process`
- How to create a small HTTP server using Node.js and Express.js
- How to create advanced routes with Express.js

## 🛠️ Technologies

- Node.js
- Express.js
- Babel
