## 📖 Description

This project covers async generators and async comprehensions in Python. You will learn how to write async generators, use async comprehensions, and type-annotate generators.

## 🎯 Learning Objectives

- How to write an asynchronous generator
- How to use async comprehensions
- How to type-annotate generators

## 🛠️ Technologies

- asyncio
