## 📖 Description

This project covers Redis basics with Python. You will learn to use Redis for basic operations, store and retrieve data, use Redis as a simple cache, and implement decorators to count and track function calls.

## 🎯 Learning Objectives

- How to use Redis for basic operations
- How to use Redis as a simple cache
- How to store and retrieve data with Redis using Python
- How to use decorators to track function calls and history

## 🛠️ Technologies

- Redis
- Python 3
- redis-py
